## Alerting System — Disambiguation

This is a disambiguation page, not a substantive technical document. It identifies three distinct meanings of "alerting system" and links to their respective articles. There is minimal technical content to extract.

## Key Concepts

- **Notification system** — software + hardware that delivers messages to a set of recipients
- **Alert messaging** — machine-to-person communication that is important or time-sensitive
- **Emergency communication system** — any system organized primarily for communicating emergency information
- These three concepts are related but distinct; the term "alerting system" is ambiguous without context
