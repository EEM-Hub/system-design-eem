- [Notification system](./Notification_system), a combination of software and hardware that provides a means of delivering a message to a set of recipients
- A system that allows [alert messaging](./Alert_messaging), machine-to-person communication that is important or time sensitive
- [Emergency communication system](./Emergency_communication_system), any system organized for the primary purpose of supporting communication of emergency information
